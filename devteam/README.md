# Dev Team

Este diretorio organiza as skills cujo uso predominante e execucao tecnica, entrega, revisao, observabilidade e artefatos operacionais de engenharia.

As skills reais continuam em `skills/`. Este diretorio existe apenas para descoberta por perfil.

## Skills desta pasta

- [`jira-tasks`](../skills/jira-tasks/SKILL.md): criacao de tasks tecnicas no Jira a partir de decomposicao local ja estruturada.
- [`github-pr-comment-triage`](../skills/github-pr-comment-triage/SKILL.md): triagem de comentarios de PR com classificacao de acao e resposta rastreavel.
- [`pull-request`](../skills/pull-request/SKILL.md): geracao, revisao e publicacao de PRs com base em diff, commits e contexto do repositorio.
- [`semantic-commit`](../skills/semantic-commit/SKILL.md): sugestao e validacao de mensagens de commit semantico com Conventional Commits.
- [`recursive-review-bugfix`](../skills/recursive-review-bugfix/SKILL.md): ciclo iterativo de review critica e bugfix ate aprovacao final.
- [`postman-collection-generator`](../skills/postman-collection-generator/SKILL.md): geracao de collections Postman a partir do codigo da API.
- [`otel-grafana-dashboards`](../skills/otel-grafana-dashboards/SKILL.md): geracao de dashboards Grafana para servicos Go com OpenTelemetry.
- [`otel-hybrid-dashboard-blueprint`](../skills/otel-hybrid-dashboard-blueprint/SKILL.md): blueprint de observabilidade hibrida Grafana + Coralogix para times SRE e plataforma.
- [`tech-debt-register`](../skills/tech-debt-register/SKILL.md): registro estruturado de debito tecnico com confronto de codebase (local ou via gh) e clarificacao em oito eixos (natureza, sintoma, localizacao, blast radius, severidade, urgencia, estrategia de pagamento, esforco).
