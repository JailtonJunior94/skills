# Geral

Este diretorio organiza as skills transversais, uteis para product managers e dev teams quando o trabalho envolve comunicacao tecnica, publicacao de mudancas ou melhoria de prompts.

As skills reais continuam em `skills/`. Este diretorio existe apenas para descoberta por perfil.

## Skills desta pasta

- [`prompt-enricher`](../skills/prompt-enricher/SKILL.md): enriquecimento de prompts para tornalos mais robustos, compactos e operacionais.
- [`github-diff-changelog-publisher`](../skills/github-diff-changelog-publisher/SKILL.md): geracao de changelog a partir de diff do GitHub com base em evidencia.
- [`github-release-publication-flow`](../skills/github-release-publication-flow/SKILL.md): fluxo completo entre leitura de diff, geracao de changelog e publicacao aprovada.
- [`confluence-changelog-publisher`](../skills/confluence-changelog-publisher/SKILL.md): publicacao controlada de changelog e resumo tecnico no Confluence.
