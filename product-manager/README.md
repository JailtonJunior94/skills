# Product Manager

Este diretorio organiza as skills cujo uso predominante e discovery, backlog, refinamento de escopo e consolidacao de artefatos de produto.

As skills reais continuam em `skills/`. Este diretorio existe apenas para descoberta por perfil.

## Skills desta pasta

- [`epic-story-discovery`](../skills/epic-story-discovery/SKILL.md): discovery estruturada de epico e user stories com perguntas obrigatorias, KPIs, trade-offs e edge cases antes de qualquer publicacao externa.
- [`us-to-prd`](../skills/us-to-prd/SKILL.md): consolidacao de contexto de uma user story do Jira para iniciar a criacao de PRD com menos ruido operacional.
- [`azure-devops-epic-stories`](../skills/azure-devops-epic-stories/SKILL.md): publicacao de bundles de discovery no Azure DevOps como epico e user stories vinculadas, com validacao, dry-run e audit log.
